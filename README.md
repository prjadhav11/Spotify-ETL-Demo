# Music Analytics Data Pipeline

## Overview
An end-to-end data engineering project that scrapes music chart data, processes CSV files using Python, stores data locally and loads analytics-ready data into MySQL for insights.

## Architecture
Spotify Charts Website  
→ Python Web Scraping  
→ Raw CSV  
→ Data Cleaning & Validation  
→ MySQL Analytics Database  
→ SQL Insights  

## Tech Stack
- Python (Requests, BeautifulSoup, Pandas)
- MySQL
